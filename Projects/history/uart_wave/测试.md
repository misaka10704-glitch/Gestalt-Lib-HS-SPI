# uart_wave：上位机波形 ↔ FPGA 回传

当前工程为 **回波模式**：PC 下发复杂波形帧，FPGA 校验后原样回传，上位机对比绘图。

## 链路

```
host 生成波形 → UART TX
       ↓
FPGA uart_rx 收帧 → 缓冲 → uart_tx 回传同帧
       ↓
host 收帧 → 与发送曲线对比
```

## 帧格式（双向相同）

```
AA 55 | SEQ | N | DATA[0..N-1] | XOR
```

- `N` ≤ 128  
- `XOR = SEQ ^ N ^ DATA[…]`（不含 `AA 55`）  
- 波特率 115200；默认 `CLK_FREQ=50MHz`

## 管脚

| 信号 | 脚 |
|------|-----|
| sys_clk | T7 |
| key[0] | D11 复位 |
| key[1] | F10 清除 LED 粘滞 |
| uart_tx | F12 |
| uart_rx | F13 |
| led[0] | R9 收发活动 |
| led[1] | R7 心跳 |
| led[2] | N6 曾收妥 |
| led[3] | P7 曾 XOR 失败 |

避开 SSPI：`C10`/`T10` 勿用。

## 上板

1. 打开 `uart_wave.gprj`，顶层 `uart_wave`，含 `uart_rx.v` / `uart_echo_core.v`
2. 综合下载（确认 `uart_rx`=F13）
3. 串口用 **`/dev/cu.usbserial-20230814131`**（你板子上有数据的那个）

```bash
cd host
python3 uart_wave_echo.py --port /dev/cu.usbserial-20230814131 --baud 115200
```

应弹出窗口：TX（主机复杂波）与 RX（FPGA 回波）重合。

旧三角波自发推流：源码仍保留 `uart_wave_core.v` / `triangle_gen.v`，需要时可改顶层例化回去。
