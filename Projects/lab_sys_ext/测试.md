## 为什么杂波「前后帧完全不像」？

**这是正常的，不是画错。**  
每一帧 UART = 一次独立 SPI 捕捉（中间有 SPI/UART 空隙）。  
PLL 失败时每次收错的比特不同 → `corr≈0`，`a`/`d` 换快照会像「完全无关系」。  
要看连续正弦：切回 **SYS**（`spi_err` 小）。

## xlen

| | |
|--|--|
| ≤128 | 单帧，X=帧内 µs，铺满 |
| 256/512… | 多帧**横排**（样点时间轴变长），帧间 **红色竖线** = UART 帧界 |
